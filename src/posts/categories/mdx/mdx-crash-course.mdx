## About MDX

[MDX](https://mdxjs.com/docs/what-is-mdx/) is a markdown extension that allows us to make use of jsx, javascript import and export statements inside marckdown documents.

```md 
# Hello, world!

<div className="note">
  > Some notable things in a block quote!
</div>
```
